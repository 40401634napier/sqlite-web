`sqlite-browser` is a web-based SQLite database browser written in Python.

Project dependencies:

* [flask](http://flask.pocoo.org)
* [peewee](http://docs.peewee-orm.com)

Usage:

```sh
$ python sqlite_browser.py /path/to/database.db
```
